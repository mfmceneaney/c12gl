Welcome to c12gl's documentation!
===================================

**c12gl** is a Python library for training and evaluating 
Graph Neural Networks in Pytorch for implementation at CLAS12.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
